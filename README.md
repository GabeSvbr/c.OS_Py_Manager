# c.OS_Py_Manager
This project began as a simple deep dive into Python's os module and grew into a complete automation "central" for CachyOS. It features a curated selection of scripts to automate software installation, perform all-in-one system updates and making system deployment fast and reproducible.

In order for it to work as it is supposed to place the python script "update.py" in the ~/home/  directoty
